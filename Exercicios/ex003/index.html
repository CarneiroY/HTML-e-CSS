Vamos aprender a como colocar imagens na pagina.

Imagem grande demais?

Normalmente podemos diminuir com o CSS o tamanho que a imagem irá aparecer na tela, porém, isso não fara com que o peso do arquivo diminua, então devemos diminuir nós mesmos utilizando um Photoshop. Normalmente 250x250px já está perfeito.

Aqui nós temos 2 exemplos de imagem, uma está em 1200x1200 e outra 250x250. Ambas possuem seu peso distinto uma da outra. A maior é pesada.

Códigos para colocar imagem no site:

Esse é o seguinte código para colocar a imagem na pagina:


    img src="logoCSSfull.png" alt="Logo do CSS"

Vale destacar aqui, que essa imagem que está sendo carregada está junto ao arquivo index.html dos site, a outra maneira de colocar imagem só que em outra pasta.

Colocando imagem de outra pasta:

Literalmente é só escrever antes do nome da imagem, a pasta onde está guardada a foto. Exemplo: arquivo/nome da imagem.formato


    <img src="imagens/logoCSSfull.png" alt="Logo do CSS">

Colocando imagens externas

ok, aprendemos a colocar imagens de duas maneiras, mas existe outra também, usando imagens extarnas, de outro servidor. Para isso devemos pegar a URL da imagem completa e colocar no atributo "SRC"
